## 🔐 JWT身份验证系统实现方案

### 核心功能
- JWT Token + Refresh Token 认证机制
- 登录/登出/刷新功能
- 记住我功能
- 路由守卫保护

### 后端实现（Django）
1. 安装 `djangorestframework-simplejwt`
2. 配置 JWT 设置（访问令牌15分钟，刷新令牌7天）
3. 创建登录/刷新/登出 API 端点
4. 配置 HttpOnly Cookie 存储刷新令牌
5. 添加登录失败限制（防暴力破解）
6. 配置 Cookie 安全属性（HttpOnly, SameSite）

### 前端实现（Vue 3）
1. 创建登录页面（用户名/密码/记住我）
2. 使用 Pinia 管理用户状态
3. JWT 存储在内存（防XSS）
4. 路由守卫控制访问权限
5. HTTP 拦截器自动刷新 Token
6. 多标签页登录状态同步

### 安全措施
- 密码 bcrypt 哈希
- HttpOnly + SameSite Cookie
- 登录失败限制（5次失败锁定15分钟）
- 登录日志记录

1. 后端 JWT 配置和API

3. 路由守卫和HTTP拦截器
